# API Contracts

**Feature**: OnesiBox Management
**Status**: N/A - No new API endpoints

This feature is a Filament admin panel enhancement only. It does not introduce new public API endpoints.

## Existing Endpoints Used

The tokens generated by this feature are used to authenticate requests to existing OnesiBox API endpoints:

- `POST /api/v1/heartbeat` - Device heartbeat
- `GET /api/v1/commands` - Fetch pending commands
- `POST /api/v1/commands/{uuid}/execute` - Mark command executed
- `POST /api/v1/playback` - Report playback events

See `routes/api.php` for full API specification.
